# CS370_Mod8

# Briefly explain the work that you did on this project: What code were you given? What code did you create yourself?
For this project I was given starter code including the maze environment, intelligent agent movement, rewards and penalties were monitored, status of the game, and functions that could test if the intelligent agent successfully reached the treasure. I implemented a deep q-learning algorithm that selects actions using the epsilon-greedy strategy and stores experiences from each episode to the neural network. 

# What do computer scientists do and why does it matter?
Computer scientists analyze a problem and find solutions through software systems and algorithms. From designing to developing a solution a computer scientist improves everyday life by creating the technology that can solve complex problems.  

# How do I approach a problem as a computer scientist?
As a computer scientist, I approach a problem by first analyzing what is asked of me. The research phase is the most important phase in the process as it sets me up to be successful in creating a solution. The research phase means I have to identify what the problem is, what the requirements are, and what is the end goal. The next phase is designing whether that is creating a paper prototype or a digital one. Then the development and testing phase occurs, this phase is trial and error, but I find working in smaller segments and breaking down the development process to make it much more manageable. 

# What are my ethical responsibilities to the end user and the organization?
My ethical responsibilities is to create software that is safe, functional, and reliable. The biggest responsibility is ensuring that anything created protects user data and privacy as well as meet the guidelines of modern regulations around personal data handling.
